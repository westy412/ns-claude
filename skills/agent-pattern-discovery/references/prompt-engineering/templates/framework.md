## Template Field Guidance

### Framework Structure
- Show the exact tag/section structure
- Use the actual XML tags or markdown structure you use
- Make it copy-pasteable

### Section Breakdown
- Explain WHY each section exists
- Give concrete examples
- Note what's optional vs required

### System vs User Message
- Be explicit about what goes where
- Note any content that could go in either
- Explain the reasoning

### Variables
- List all placeholders
- Show expected format
- Note any transformations needed
